# Marmosa Documentation

This folder contains project documentation for the Rust port.

## Key documents

- `dcb-specification.md` and `dcb-projections.md`: DCB concepts mapped to Marmosa APIs.
- `specifications-index.md`: migrated spec status and dependency order.
- `use-cases.md`: scenario fit guidance for file-system event storage.
- `durability-quick-reference.md`: durability priorities and operational guidance.
- `release-process.md` and `release-quick-reference.md`: release workflow.

## Principles

- Keep docs aligned with actual Rust APIs.
- Prefer concise examples that compile or are clearly marked `no_run`/`ignore`.
- Treat `/docs` as the single place for high-level project docs.
